## 项目使用说明(本项目后端已废弃，不再维护)
1. 安装node.js
2. 下载项目并解压
3. 在终端(项目根目录中)输入以下代码
```
npm install
npm run build
```
4. 打开dist中的index.html文件即可正常使用
